🌌 About

Mentora AI is an offline-first web application that helps students explore career paths through an interactive constellation of skills. Users select a career goal and complete 10 skill stars in sequence to build their personalized learning journey.

✨ Features

- Career goal selection (6 predefined paths + custom goal)
- Interactive 10-skill constellation
- Sequential skill progression
- Gold glow for completed stars
- Rocket milestone celebrations after stars 0, 2, 4, 6, and 8
- Graduation cap + confetti after all 10 stars
- Mentor tips and guidance
- Progress saved using IndexedDB (localStorage fallback)
- Fully offline and responsive

🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- IndexedDB

▶️ Run Locally

1. Clone or download the project.
2. Open "index.html" in any modern browser.
3. Select a career goal.
4. Complete stars in order and track your progress.

📁 Files

- "index.html" – Main application
- "style.css" – Styling and responsiveness
- "script.js" – Constellation logic, storage, milestones
- "README.md" – Documentation

💡 Innovation

Mentora AI combines career guidance, gamification, and offline accessibility to create an engaging learning experience. The architecture is designed to support future AI-powered roadmap generation.
